## Run Locally

Clone the project

```bash
  git clone https://github.com/raihan71/my-edu
```

Go to the project directory

```bash
  cd my-edu
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React, ReactQuery, TailwindCSS, Formik

**Server:** Node, Express
